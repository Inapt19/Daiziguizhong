# 每日一题 180604

最大前辍和

给定一个长度为n的序列a[1..n]，现在你可以进行最多k次操作，每次操作能交换序列中任意两个数，要求最大化最大前缀和的值。

最大前缀和的定义：

![](http://wizmann-pic.qiniudn.com/18-5-31/40662330.jpg)

## 输入

第一行两个非负整数n,k，1 ≤ n ≤ 50000，1 ≤ k ≤ 3。

第二行n个整数，第i个整数表示ai   (|ai| ≤ 109)

## 输出

输出一个非负整数，表示最大的最大前缀和的值。

## 样例输入
```
5 1
-1 -3 2 3 2
```

## 样例输出
```
6
```
