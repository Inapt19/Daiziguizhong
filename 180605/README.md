# 专题讨论：牛客练习赛19（1）

本场比赛大概是CF Div2水平的（甚至要更简单一点），总体难度不高。不过有坑题和题目描述不清的问题（小平台比赛的通病）。

这里说的坑题，就是出题者故意在题面中使用反直觉或反常识的模糊描述来误导做题人。这种题对算法水平的提高没有帮助，所以我个人对这种题是有一些意见的。

对这场比赛给出一个评级（个人意见）：

```
题目难度：★★★
题面描述：★★★
推荐程度：★★★☆
```
