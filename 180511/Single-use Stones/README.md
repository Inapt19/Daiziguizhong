每日一题 180511

Single-use Stones

一群青蛙想过一条宽度为W的河（可以看做一条线段），每只青蛙最多能跳L格。（1 <= l < w <= 1e5)

河上有一些石头，位于线段的整点上，每个整点有a[i]个石头。每个石头只能被一只青蛙踩一次。(0 <= a[i] <= 1e4)

问最多能有多少只青蛙可以跳过河流。

sample input      
10 5       
0 0 1 0 2 0 0 1 0

sample output       
3

解释：一只青蛙可以从3->8跳到对岸，还有两只青蛙可以从第5块石头上跳到对岸

原题链接：http://codeforces.com/contest/965/problem/D
